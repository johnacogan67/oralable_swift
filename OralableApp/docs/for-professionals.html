For researchers, therapists, and clinicians. View participant data. Track progress. Enhance care.

One Platform, Multiple Applications

Oralable for Professionals is designed for healthcare providers, researchers, and therapists who need objective muscle activity data. Whether you're conducting research, providing therapy, or monitoring clenching and grinding, Oralable provides the tools you need.

Data Export

Export data in CSV format for analysis, research documentation, or integration with other systems.

Important: Wellness Data Only

Oralable is a consumer wellness device, not a medical device. The data should be used to support conversations and awareness, not for diagnosis or treatment decisions. Always rely on clinical examination and professional judgment.

How It Works

  1. Participant Uses Oralable: Your participant records muscle activity sessions using the Oralable app
  2. Participant Generates Share Code: In the app, they generate a 6-digit share code
  3. You Enter the Code: In Oralable for Professionals, you enter the code to connect
  4. View Their Data: You can now view their shared muscle activity patterns
  5. Participant Controls Access: Participants can revoke access at any time
